# Agent Orchestration System

## Overview

The Agent Orchestration system allows you to coordinate multiple AI agents to work together in three distinct patterns: Parallel, Chain, and Consensus. All orchestrations are created and managed manually through the UI.

## Orchestration Types

### 1. Parallel Orchestration
Execute multiple agents simultaneously for independent tasks.

**When to use:**
- Tasks that don't depend on each other
- Need for faster processing through concurrency
- Generating multiple outputs at once
